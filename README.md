# WEEK-3-SOC-REPORT
# pfSense Firewall Integration with Wazuh SIEM

## 📌 Project Overview

This project demonstrates the integration of **pfSense Firewall** with **Wazuh SIEM** to build a centralized security monitoring environment. The objective was to collect firewall logs, forward them to the Wazuh Manager, create custom detection rules, and monitor security events through the Wazuh Dashboard.

The project was completed in a virtual lab using VMware and provided practical experience in firewall management, log analysis, SIEM integration, and incident response.

---

## 🎯 Objectives

- Deploy and configure pfSense Firewall
- Install and configure Squid Proxy Server
- Configure Remote Syslog for centralized logging
- Integrate pfSense logs with Wazuh SIEM
- Create custom Wazuh decoders and detection rules
- Validate alerts using Wazuh Logtest
- Monitor security events through the Wazuh Dashboard
- Develop an Incident Response Plan
